# AufgabenMathe

LaTeX-Skript mit Aufgaben und Lösungen zur Vorbereitung auf das niedersächsische Abitur (Mathematik).

## Inhalt

- Themenbereich Analysis (inkl. Exponential- und Logarithmusfunktionen)
- Aufgaben- und Lösungsblöcke mit automatischen Querverweisen
- Einheitliche Nummerierung nach Kapitel/Abschnitt/Unterabschnitt

## Voraussetzungen

- TeX Live (oder eine vergleichbare LaTeX-Distribution)
- `pdflatex`

## Kompilieren

```bash
pdflatex -interaction=nonstopmode -halt-on-error Mathe_Aufgaben.tex
pdflatex -interaction=nonstopmode -halt-on-error Mathe_Aufgaben.tex
